# E-Voting System

A secure and database-driven web-based voting system developed using PHP, MySQL, HTML, CSS, and JavaScript. The system provides a transparent digital voting environment with separate Admin and User panels for efficient election management and secure vote handling.

---

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

---

# System Modules

## User Panel

Users can securely participate in the election process through an interactive dashboard.

### Features
- Secure user authentication
- Digital vote casting system
- Candidate information display
- Voting status verification
- Election result viewing
- Responsive and user-friendly interface

---

## Admin Panel

The admin panel provides complete control over election management and system operations.

### Features
- Add and manage candidates
- Start and stop elections dynamically
- Monitor voting activity
- Declare election results
- Conduct and manage new elections
- Manage overall voting records

---

# System Highlights

- Secure authentication and access control
- Real-time vote storage and result generation
- Database-driven architecture
- Interactive frontend design
- Organized admin and user dashboards

---

# Future Improvements

- OTP/Email verification
