# E-Voting System

A secure web-based voting system developed using PHP and MySQL with HTML, CSS, and JavaScript for the frontend. The system provides transparent and efficient digital voting with separate Admin and User panels.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Features

### User Panel
- Secure user authentication
- Cast votes digitally
- View candidates and election results
- Check voting status and confirmation after voting

### Admin Panel
- Manage candidates and voting data
- Start and stop elections dynamically
- Declare election results
- Conduct and manage new elections
- Monitor overall voting activity

## System Highlights
- Database-driven architecture
- Real-time vote storage and result generation
- Secure and organized voting management
- Interactive and responsive web interface
